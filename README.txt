	killer/ConsoleApplication1 - В данной ветке представлена утилита, которая мониторит процессы Windows и "убивает" те процессы, которые работают слишком долго. 
- На входе три параметра: название процесса, допустимое время жизни (в минутах) и частота проверки (в минутах).
- Утилита запускается из командной строки. При старте она считывает три входных параметра и начинает мониторить процессы с указанной частотой. Если процесс живет слишком долго – завершает его и выдает сообщение в лог (на рабочий стол).

	selen - Ветка, для задания с Selenium WebDriver. Утилита имеет 2 режима:
- Запуск с заранее заданными параметрами (из задания)
- Запуск с задаваемми параметрами
